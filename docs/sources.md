# Использованные источники

- **Официальная документация Python по модулю argparse**  
  Основной источник формальных сведений о возможностях модуля: класс `ArgumentParser`, методы `add_argument`, `add_subparsers`, работа с `choices`, `nargs`, `fromfile_prefix_chars`, примеры использования подкоманд и форматирования справки.

- **Документация по стандартной библиотеке Python (общие разделы)**  
  Использовалась для уточнения деталей работы типов `pathlib.Path`, обработки исключений и соглашений по структуре CLI‑программ.

- **Статьи и учебные материалы о создании CLI на Python (онлайн‑блоги, учебники)**  
  По ним были переняты общие приёмы проектирования интерфейса командной строки: выделение подкоманд, использование взаимно исключающих флагов, разбиение кода на парсер и обработчики, а также рекомендации по оформлению `--help` и примеров запуска.

- **Сравнительные обзоры библиотек argparse / click / typer**  
  Помогли сформировать понимание сильных и слабых сторон `argparse` по сравнению с более высокоуровневыми решениями и чётко сформулировать ограничения инструмента в разделе выводов.

- **Собственные эксперименты с консольным запуском программы argparse_demo.py**  
  На практике проверялись различные комбинации аргументов и подкоманд, сценарии ошибок и некорректного ввода, чтобы убедиться, что поведение соответствует ожиданиям и может служить учебным примером.
