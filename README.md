## Argparse Demo

Обучающая консольная программа на Python, демонстрирующая основные возможности стандартного модуля `argparse`:
позиционные и опциональные аргументы, подкоманды, группы взаимоисключающих флагов, чтение аргументов из файла и базовую обработку ошибок.

### Цель

Дать практическое понимание того, как проектировать интерфейс командной строки и как использовать ключевые функции `argparse` на реальном примере.

### Что умеет программа

- **Подкоманда `greet`**: выводит приветствие, поддерживает повторения и режим "крика" (`--shout`).
- **Подкоманда `calc`**: выполняет простые арифметические операции (`add`, `sub`, `mul`, `div`) над списком чисел, демонстрирует `choices`, `nargs`, группы взаимоисключающих флагов (`--int` / `--precise`).
- **Подкоманда `file stats`**: считает количество строк, слов и символов в текстовом файле, демонстрируя работу с типом `Path` и обработку ошибок.
- **Глобальные флаги**: `--verbose` (подробный вывод) и `--output` (сохранение результата в файл).
- **Разбор аргументов из файла**: поддержка `fromfile_prefix_chars="@"` — аргументы можно передать из файла `@args.txt`.

### Установка и запуск

1. Убедитесь, что установлен Python 3.10+.
2. Перейдите в папку проекта:

```bash
cd C:\Users\studentcoll\Desktop\Prac
```

3. Запустите программу:

```bash
python argparse_demo.py --help
```

Примеры:

```bash
python argparse_demo.py greet Alex --times 3 --shout
python argparse_demo.py calc add 1 2 3 4 --precise
python argparse_demo.py file stats example.txt --verbose
```

Пример использования файла аргументов (Windows, PowerShell / CMD):

1. Создайте файл `args.txt` рядом с программой, например:

```text
greet
Alex
--times
2
```

2. Запустите:

```bash
python argparse_demo.py @args.txt
```
